# Лабораторная работа №2: NLP (Задание на Пятерку с плюсом)

## Бизнес-задача
Приют для животных. Разработан Proof-of-Concept прототип, использующий LLM + RAG (поиск по FAISS) для оценки сроков усыновления животного и рекомендаций по улучшению его профиля.

## Стек
- Docker & Docker Compose
- FastAPI (Обертка над LLM)
- Ollama + Qwen2.5:0.5B
- FastMCP
- `sentence_transformers` + `faiss` (для тулы ретривера RAG)

## Инструкция по запуску (Воспроизводимость)

1. Убедитесь, что у вас установлен Docker и Docker Compose.
2. Откройте терминал в корневой папке проекта.
3. Соберите и запустите контейнеры (сборка займет несколько минут, так как скачивается модель Qwen и устанавливаются зависимости):
   ```bash
   docker compose up --build -d
